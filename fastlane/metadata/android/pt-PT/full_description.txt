Esta aplicação permite verificar os requisitos para o GSI Treble e determinar o tipo correto de GSI para o seu dispositivo.

Elementos chave:
* Verifica o suporte para o Treble
* Verifica a versão Treble/VNDK
* Verifica a arquitectura de sistema
* Verifica se é A-only ou A/B
* Procura por binder64
* Procura por System-as-root
* Não necessita de acesso Root
* Suporta tema escuro e MD3

Pode fazer download de GSIs na Internet. Visite o fórum XDA (não relacionado connosco) para alguns tutoriais fáceis.

Código fonte: https://gitlab.com/hackintosh5/TrebleInfo
